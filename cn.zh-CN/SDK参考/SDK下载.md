# SDK下载 {#concept_71770_zh .concept}

本文为您介绍了音视频通信客户端SDK的相关文档链接，帮助您快速了解并使用音视频通信相关功能。

**说明：** 目前音视频通信OpenAPI支持版本：Java、Python、PHP、Go、.NET、Node.js。获取更多版本SDK，请单击[阿里云SDK中心](https://developer.aliyun.com/tools/sdk)。
